# stable-trade-binance
Este é um bot para realizar trade nos pares USDT da corretora de criptomoeda [Binance](https://www.binance.com).

**O bot ainda está em fase de desenvolvimento.**

A ideia principal é identificar um melhor preço para compra/venda nos pares USDT e realizar as ordens automaticamente por meio da API.

Para cada ordem que for executada pelo bot, um e-mail é enviado com as informações do mercado naquele momento. Os e-mails enviam as informações referente ao mercado e o valores de compra/venda que foram estipulados e considerados pelo bot.
